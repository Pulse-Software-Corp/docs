## Overview

The Pulse Platform consists of two web applications:

<CardGroup cols={2}>
  <Card title="Console" icon="gauge" href="https://console.runpulse.com">
    Manage your organization, API keys, usage, and billing.
  </Card>
  <Card title="Playground" icon="browser" href="https://platform.runpulse.com">
    Upload documents, build extraction pipelines, and preview results in a visual UI.
  </Card>
</CardGroup>

## Pipelines

Choose the right pipeline for your use case — each one builds on the previous:

<CardGroup cols={3}>
  <Card title="Extract" icon="file-lines" href="/platform-reference/extract">

  <Card title="Extract → Schema" icon="table" href="/platform-reference/extract-schema">
