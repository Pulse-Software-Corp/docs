| Field | Replacement |

| `schema` | Use `structuredOutput.schema` instead |
| `schemaPrompt` | Use `structuredOutput.schemaPrompt` instead |
| `experimentalSchema` | Use `structuredOutput.schema` instead |
| `customPrompt` | No replacement |
| `thinking` | No replacement |
