### Deprecated Fields

The following fields are deprecated and will be removed in a future version:

| Field | Replacement |
|-------|-------------|
| `structured_output` | Use [`/schema`](/api-reference/endpoint/schema) endpoint after extraction. Pass `extraction_id` + `schema_config`. |
| `schema` | Use [`/schema`](/api-reference/endpoint/schema) endpoint after extraction |
| `schema_prompt` | Use [`/schema`](/api-reference/endpoint/schema) endpoint with `schema_config.schema_prompt` |
| `experimental_schema` | Use [`/schema`](/api-reference/endpoint/schema) endpoint after extraction |
| `custom_prompt` | No replacement |
| `thinking` | No replacement |
